Traceability

07 / transcript ↔ doc ↔ timestamp

This is the join that makes every AI claim checkable against the original moment — audio, words, and whatever was on screen at the time.

  • Every agent output stores the array of source_segment_ids it was derived from — never just a confidence number on its own.
  • At render time, the UI resolves segments → timestamp range → any snapshot inside that range → shows transcript, slide, and cued audio together.
  • The same join powers the Agent Activity page and the human-review queue, so a reviewer always sees evidence, not an isolated claim.
